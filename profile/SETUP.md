# Configuración del Perfil

## Cómo añadir proyectos

1. Edita `projects/projects.yml`
2. Añade tu proyecto
3. Haz push. La GitHub Action actualizará el README.

## Edición de Demos
Para que aparezcan los botones de Live Demo, asegúrate de poner la URL en el campo `demo:` del YAML.

## Badges
Puedes cambiar el estilo de los badges editando `scripts/render_projects.py`.
